Description du projet SushiFAST

L'application web SushiFAST permet à un serveur de prendre la commande d'un client dans un point de vente.
Afin d’approcher au plus d'une réalité commerciale SushiFAST utilise une application Back-end avec NodeJS dans le cadre d’une API présentant la gamme de produits à la vente.

Afin de réaliser ce projet nous avons travaillé en équipe de 2.
