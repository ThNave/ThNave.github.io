Description du projet Vetux-Line

Cette application Web permet la fusion entre 2 fichiers CSV selon un type de fusion choisi par l'utilisateur, en respectant certaines contraintes de sélection (exemple : si le client est mineur, ont le retire de la liste ...).

Afin de réaliser ce projet nous avons travaillé en équipe de 3.
